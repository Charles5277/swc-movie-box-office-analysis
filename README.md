# 陽光女子合唱團 票房分析儀表板

台灣電影《陽光女子合唱團》票房數據視覺化分析專案。

## 功能特色

- 📊 週票房趨勢追蹤
- 🏆 國產電影票房排行對比
- 📈 累計票房與觀影人次分析
- 🎯 奪冠進度預測（目標：超越《海角七號》）
- 🌙 深色模式支援

## 技術棧

- [Nuxt 4](https://nuxt.com/) - Vue 全棧框架
- [@nuxt/ui](https://ui.nuxt.com/) - UI 元件庫
- [nuxt-charts](https://github.com/nuxt-modules/charts) - 圖表視覺化
- [Cloudflare Pages](https://pages.cloudflare.com/) - 部署平台

## 開發

```bash
# 安裝依賴
pnpm install

# 啟動開發伺服器
pnpm dev

# 建置生產版本
pnpm build

# 預覽生產版本
pnpm preview
```
